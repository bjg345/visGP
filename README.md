Accompanying code for Gilbert & Datta 2023: Visibility graph-based covariance functions for scalable spatial analysis in nonconvex domains
(https://arxiv.org/abs/2307.11941)

Simulated data is generated by the "gen_" files in the main directory. The "finger_sim" directory runs various methods for the study referred to as "Fork-shaped domain" in the paper. In this codebase, "visGP" is known as "cs" for "covariance selection." After model fitting, predictions are made by various variations of the general algorithm (nearest clique ["nearest"], maximum precision ["maxprec"], precision-weighted ["weighted"], and standard kriging ["euclid"]).

The Chesapeake Bay analysis is in the "Chesapeake" directory.
